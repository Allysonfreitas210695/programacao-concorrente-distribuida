Questão 8 - Diagrama Butterfly Allgather
=========================================

Configuração: comm_sz=8, x=(0,1,2,...,15), distribuição em bloco.
Cada processo recebe 2 elementos:
  Proc 0: {0,1}   Proc 1: {2,3}   Proc 2: {4,5}   Proc 3: {6,7}
  Proc 4: {8,9}   Proc 5: {10,11} Proc 6: {12,13} Proc 7: {14,15}

O Allgather coletivo em borboleta funciona em log2(8) = 3 fases.
Em cada fase, processos parceiros trocam seus dados acumulados até então.

FASE 1 (stride=1) - pares (0,1),(2,3),(4,5),(6,7):
  Proc 0 ↔ Proc 1:  ambos ficam com {0,1,2,3}
  Proc 2 ↔ Proc 3:  ambos ficam com {4,5,6,7}
  Proc 4 ↔ Proc 5:  ambos ficam com {8,9,10,11}
  Proc 6 ↔ Proc 7:  ambos ficam com {12,13,14,15}

FASE 2 (stride=2) - pares (0,2),(1,3),(4,6),(5,7):
  Proc 0 ↔ Proc 2:  ambos ficam com {0,1,2,3,4,5,6,7}
  Proc 1 ↔ Proc 3:  ambos ficam com {0,1,2,3,4,5,6,7}
  Proc 4 ↔ Proc 6:  ambos ficam com {8,9,10,11,12,13,14,15}
  Proc 5 ↔ Proc 7:  ambos ficam com {8,9,10,11,12,13,14,15}

FASE 3 (stride=4) - pares (0,4),(1,5),(2,6),(3,7):
  Proc 0 ↔ Proc 4:  ambos ficam com {0,1,2,...,15} ✓
  Proc 1 ↔ Proc 5:  ambos ficam com {0,1,2,...,15} ✓
  Proc 2 ↔ Proc 6:  ambos ficam com {0,1,2,...,15} ✓
  Proc 3 ↔ Proc 7:  ambos ficam com {0,1,2,...,15} ✓

Visualização ASCII (Fase 1 e 2):

  P0 --[{0,1}]--><--[{2,3}]-- P1     (Fase 1)
  P2 --[{4,5}]--><--[{6,7}]-- P3
  P4 --[{8,9}]--><--[{10,11}]--P5
  P6 --[{12,13}]--><--[{14,15}]--P7

  P0 --[{0..3}]--><--[{4..7}]-- P2   (Fase 2)
  P1 --[{0..3}]--><--[{4..7}]-- P3
  P4 --[{8..11}]--><--[{12..15}]--P6
  P5 --[{8..11}]--><--[{12..15}]--P7

  (Fase 3: cada grupo de 4 troca com o grupo espelhado a distância 4)

Total: 3 fases = log2(8).
A cada fase, cada processo dobra a quantidade de dados que possui.
